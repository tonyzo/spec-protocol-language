## 📝 四类冲突检测

### 1. 术语冲突 (terminology.rs)

**检测逻辑**:
```rust
// 按SPEC参数名分组,查找同一参数对应多个不同的标准术语
GB150: "设计压力" → design_pressure
ASME VIII: "Design Pressure" → design_pressure
```

**严重度**: Info(已映射到同一SPEC参数)

**输出示例**:
```
参数 'design_pressure' 在不同标准中有不同术语: 
GB150: 设计压力, ASME_VIII: Design Pressure
```

---

### 2. 参数冲突 (parameter.rs)

**检测逻辑**:
```rust
// 按SPEC参数名分组,查找同一参数对应多个不同的定义
GB150: δ_min (最小壁厚,考虑腐蚀裕量)
ASME VIII: Required Thickness (所需厚度,由公式计算)
```

**严重度**: Warning(需要人工确认)

**输出示例**:
```
参数 'delta_min' 在不同标准中有不同定义:
  GB150 (最小壁厚): 考虑腐蚀裕量后的最小厚度
  ASME_VIII (Required Thickness): 由公式计算得到的最小厚度
```

---

### 3. 规则冲突 (rule.rs)

**检测逻辑**:
```rust
// 检测同一参数在不同标准中的单位是否一致
GB150: design_pressure → MPa
ASME VIII: design_pressure → psi
```

**严重度**: Error(要求一致,但单位不同)

**输出示例**:
```
参数 'design_pressure' 在不同标准中使用不同单位:
GB150: Some("MPa"), ASME_VIII: Some("psi")
```

---

### 4. 单位冲突 (unit.rs)

**检测逻辑**:
```rust
// 检测同一参数在不同标准中单位不同
GB150: design_pressure → MPa
ASME VIII: design_pressure → psi
```

**严重度**: Warning(需要单位转换)

**输出示例**:
```
参数 'design_pressure' 在不同标准中使用不同单位:
  GB150: Some("MPa")
  ASME_VIII: Some("psi")
解决建议: 需要单位转换: 1 MPa = 145.038 psi
```
