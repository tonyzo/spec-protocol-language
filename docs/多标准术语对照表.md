## 🔗 跨标准引用关系

### GB150 → ASME VIII

```
GB150.3-5.3 (内压圆筒)
    ↓
ASME VIII-UG-27 (Thickness of Shells Under Internal Pressure)
    ↓
公式对比:
GB150: δ = (p × D_i) / (2[σ]^t φ - p)
ASME:  t = (P × R) / (S × E - 0.6P)
```

### 单位换算流程

```
1. 识别GB150参数 (MPa, mm, °C)
   ↓
2. 转换为ASME单位 (psi, in, °F)
   ↓
3. 代入ASME公式计算
   ↓
4. 结果转换回SI单位(如需要)
```

## 💡 使用建议

### 1. 单位换算注意事项

- **压力**: 1 MPa = 145.038 psi (近似145)
- **厚度**: 1 in = 25.4 mm (精确)
- **温度**: °F = °C × 9/5 + 32 (线性变换)
- **应力**: 1 MPa = 145.038 psi

### 2. 同义词处理

ASME VIII的术语已标注中文同义词:
```json
{
  "standard_term": "Design Pressure",
  "synonyms": ["设计压力"]
}
```

### 3. 跨标准规则编写

编写跨标准规则时,需要:
- 明确单位制(SI或US Customary)
- 添加单位转换逻辑
- 使用synonyms字段标注对照术语
