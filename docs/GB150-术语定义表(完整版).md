# GB/T 150 压力容器 术语定义与概念映射表

> 本文档将GB/T 150 压力容器标准中的术语、概念与SPEC Protocol的参数/限值/属性建立映射关系。
> 
> **数据来源**: GB/T 150.1-2024 通用要求, GB/T 150.2-2024 材料, GB/T 150.3-2024 设计, GB/T 150.4-2024 制造、检验和验收
> 
> **映射原则**: 
> - 标准术语 → SPEC参数名(`parameters`/`limits`/`attrs`)
> - 明确哪些参数需计算软件提供
> - 标注术语出处条款号

---

## 4. 跨标准引用关系

### GB150.1→GB150.3
通用要求引用设计压力计算
**涉及术语**: design_pressure, calc_pressure, MAWP

### GB150.3→GB150.4
设计参数用于制造检验判定
**涉及术语**: design_pressure, delta, medium_toxicity

### GB150.1→GB150.2
通用要求引用材料温度范围
**涉及术语**: design_temperature, temp_allow_max, temp_allow_min

### GB150.3→GB150.2
设计计算引用许用应力
**涉及术语**: sigma_allow, R_m, R_eL, E_t

### GB150.4→GB150.3
制造检验引用设计参数
**涉及术语**: design_pressure, delta, test_pressure_required

## 5. 校验规则

- **所有parameters必须携带unit字段**: terms[*].spec_mapping.type=='parameter' implies spec_mapping.unit != null
- **所有limits必须携带unit字段**: terms[*].spec_mapping.type=='limit' implies spec_mapping.unit != null
- **attrs类型不能有unit**: terms[*].spec_mapping.type=='attr' implies spec_mapping.unit == null
- **术语必须有source_clause**: terms[*].source_clause != null
- **confidence必须是high/medium/low之一**: terms[*].confidence in ['high', 'medium', 'low']
