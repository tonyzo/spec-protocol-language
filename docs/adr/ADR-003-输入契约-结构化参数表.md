# ADR-003: 输入契约 —— 结构化参数表，限值外部提供

## 状态
已确认（2026-06-29）

## 决策
外部计算软件向 SPEC 输出**结构化参数表**（JSON/YAML），其中包含：

1. **参数值**：工程设计结果。如 `δ = 16 mm`、`σθ = 120 MPa`。
2. **限值**：判据边界值，同样由计算软件提供。如 `δ_min = 12 mm`、`[σ]^t = 163 MPa`。

SPEC 只做「参数 vs 限值」的匹配判定，**不计算限值**。

## 待澄清（下一轮访谈）
- 参数值与限值在参数表中如何区分？（同一扁平表 / 分区 / 命名约定）
- 是否需要单位校验（mm 与 m 混用时报错）。

## 后果
- SPEC 无需内置材料库 / 查图能力。
- 限值的正确性由计算软件负责。
- 需要明确的参数表 Schema（见后续设计）。
